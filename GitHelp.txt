Команды для регистрации ваших данных
git config --global user.name "Your name"
git config --global user.email "your-email@example.com"


Команды только для одно проекта
git init
git remote add origin https://github.com/arzy-54-2/oop-63-2.git


Основные команды для работа с Git

git status  | состаяние проекта
git add .      | Скопировать все в буффэр обмена
git commit -m "lesson1"
git branch

git branch branch_name          | Создание ветки

git switch branch_name
                                | Переклюбчение на ветку
git checkout branch_name

git switch -c branch_name
git checkout -b branch_name


git reset .
git restore .


git pull      | Стянуть изменения из репы
